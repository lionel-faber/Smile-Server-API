Drawer Example

About this

This library contains Material Design Navigation Drawer component that could be easily put in your site.

You can move drawer with touch event :)

Drawer perfectly worked on Desktop and Mobile devices.

You can add any content to drawer. Links and event(onclick, etc.) working ok.

Used Hamburger icon: Hamburger To Back Arrow

More info see in GitHub: Material-Drawer

I set "float to right" to title and icon only for example :)
